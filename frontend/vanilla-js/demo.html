ðŸ”Œ Integration with Harvey API

Use this player to render videos from Harvey's chat API responses.

// Fetch chat response from Harvey API
fetch('/v1/chat', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    message: 'Tell me about dividend stocks',
    user_id: 'user123'
  })
})
.then(res => res.json())
.then(data => {
  // Check if videos were recommended
  if (data.video_metadata && data.video_metadata.length > 0) {
    // Render each video with HeyDividend player
    data.video_metadata.forEach((video, index) => {
      const player = new HeyDividendPlayer(
        `player-${index}`,
        video,
        { variant: 'expanded' }
      );
    });
  }
});
